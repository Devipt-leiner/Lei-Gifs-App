Developed by Leiner Barrios

This is a small application developed in Angular that allows you to search for all kinds of gifs by consuming the GIPHY Developers API.

The sidebar stores the history of recent searches limiting the amount to ten, also allowing you to request the search again by clicking one of the items. The results section is designed using flex-wrap to be responsive on all devices and Bootstrap cards are used to give the results a visual touch.
